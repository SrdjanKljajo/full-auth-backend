One To Meny Full Auth

Node backend API with security config and one to meny association - postgreSQL and sequelize ORM. API is posted on AWS with EC2 instances and RDS database.

The API consists of two related models. User model and Post model. Api can be tested using the Postman app.

A route can be used to get all posts:

GET http://3.123.21.234/api/v1/posts

A route can be used to get single post:

GET http://3.123.21.234/api/v1/posts/:slug
To create a post by yourself, you need to register. It is necessary to enter a valid email address to which you will be sent a token to activate your account. After activation, you will be assigned a user role that includes certain privileges. If you have logged out in the meantime, you can log in again with the password and email address you entered. If you forget your password, there is a reset option.
